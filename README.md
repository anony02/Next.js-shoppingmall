# Project Name

## Installation

1. cd shoppingmall

2. npm install

3. npm run dev

3. `/cart`: 장바구니 페이지
4. `/detail/:id`: 상품별 상세 페이지
5. `/:category`: 카테고리별 페이지
6. `/kakao/oauth`: 카카오 간편로그인 페이지
